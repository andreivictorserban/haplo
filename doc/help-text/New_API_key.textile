h1. New API key

Click the _Create new API key_ button to create an API key, then _Reveal API key_ to show the API key you created.

API keys are equivalent to a username and password combination. Treat them as strictly confidential.

If you would like to write some software which communicates with Haplo, you can find the API documentation <a href="https://docs.haplo.org/dev" target="_blank" rel="noopener">in our developer documentation</a>.
